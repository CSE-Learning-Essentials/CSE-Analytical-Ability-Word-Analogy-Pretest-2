SECTION 2: ANALYTICAL ABILITY

PART 1: WORD MEANING

10-ITEM TEST | PASSING SCORE: 8

Directions: Each question below consists of a related pair of words. From among the suggested answers, choose the word or group of words that best expresses a similar relationship from the original pair.

1. Professor : Teacher :: ______ : Journalist

2. ______ : Words :: Directory : ______

3. Court Trial : Evidence :: Medical Trial : ______

4. Plant : Oxygen :: ______ : ______

5. Compass : Direction :: Car : ______

6. Protection : Endangered Species :: ______ : ______

7. Dove : Peace :: Scale : ______

8. Chapter : Book :: ______ : ______

9. Endemic : Restricted :: Pandemic : ______

10. Girth : Circumference :: ______ : ______

Answer Explanations:

  1. E. columnist
    Relationship: Person to Role.
    Professor : Teacher :: Columnist : Journalist
    Explanation: A professor is a type of teacher; a columnist is a type of journalist. Both pairs show a general-to-specific role relationship.
  2. A. Lexicon : Listings
    Relationship: Object to Content / Collection to Items.
    Lexicon : Words :: Directory : Listings
    Explanation: A lexicon is a collection of words. A directory is a collection of listings. Both pairs show an object-to-content relationship.
  3. C. result
    Relationship: Event to Basis/Proof.
    Court Trial : Evidence :: Medical Trial : Result
    Explanation: A court trial relies on evidence, while a medical trial relies on results. Both pairs show event-to-evidence relationships.
  4. E. animals : carbon dioxide
    Relationship: Living Being to Gas Exchange.
    Plant : Oxygen :: Animals : Carbon Dioxide
    Explanation: Plants produce oxygen; animals produce carbon dioxide. Both pairs show a living being and its related gas exchange.
  5. D. transportation
    Relationship: Tool/Instrument to Purpose.
    Compass : Direction :: Car : Transportation
    Explanation: A compass is used to find direction; a car is used for transportation. Both pairs show instrument to intended purpose.
  6. D. restoration : historical building
    Relationship: Action to Purpose / Protection to Beneficiary.
    Protection : Endangered Species :: Restoration : Historical Building
    Explanation: Protection is done to safeguard endangered species. Restoration is done to preserve historical buildings.
  7. D. fairness
    Relationship: Symbol to Concept.
    Dove : Peace :: Scale : Fairness
    Explanation: A dove symbolizes peace; a scale symbolizes fairness. Both pairs link a symbol to the concept it represents.
  8. B. stanza : poem
    Relationship: Part to Whole.
    Chapter : Book :: Stanza : Poem
    Explanation: A chapter is part of a book; a stanza is part of a poem. Both pairs show part-to-whole relationships.
  9. A. boundless
    Relationship: Condition to Extent.
    Endemic : Restricted :: Pandemic : Boundless
    Explanation: Endemic is restricted to a specific area; pandemic spreads without boundary (boundless). Both pairs show condition-to-extent relationships.
  10. B. foot : inch
    Relationship: Synonymous Measurement Terms.
    Girth : Circumference :: Foot : Inch
    Explanation: Girth is another word for circumference; foot is a unit related to inch. Both pairs show synonymous or related measurement terms.
